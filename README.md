# Go API 

## CRUD operations with GORM

- Create (POST)
- Read (GET)
- Update (PUT)
- Delete (DELETE)

## API calls:
-   / 
![](images/homepic.jpeg)
- 	/users -> GET (All Users)
-	/user/{name}" -> "DELETE
-	/user/{name}/{lastname}/{city}/{email} -> PUT
-	/user/{name}/{lastname}/{city}/{email} -> POST
